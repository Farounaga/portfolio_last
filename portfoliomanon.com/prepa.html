Préparation des Postes Clients

1. Installation de l’Antivirus ESET

Pour sécuriser les postes clients, nous installons et configurons ESET :

  • Téléchargement du package d’installation depuis le site officiel.
  • Lancement de l’installation avec les paramètres recommandés.
  • Activation de la licence.
  • Configuration des règles de sécurité (analyse en temps réel, mise à jour automatique).

2. Configuration de Base du Poste

Une fois l’antivirus installé, voici les configurations de base effectuées :

  • Mise à jour du système : Installation des dernières mises à jour Windows/Linux.
  • Création d’un compte utilisateur : Configuration d’un compte utilisateur standard.
  • Installation des logiciels essentiels : Suite Office, navigateur web, outils spécifiques du client.
  • Configuration du réseau : Connexion au domaine, paramétrage des IP et DNS si nécessaire.

3. Tests et Validation

Une fois la configuration terminée, nous effectuons :

  • Un test de connexion au réseau et aux serveurs.
  • Un scan antivirus complet pour vérifier qu’aucune menace n’est présente.
